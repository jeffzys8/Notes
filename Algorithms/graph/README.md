# 基本概念

## 图的存储方式

- **邻接表** ：使用链表构成的的一维数组```[]LinkedList<int>```存储整个表，其中每一条链表存储对应结点的全部邻居

- **邻接矩阵**：使用二维数组```[][]int```存储整个表，其中```[i][e]```表示i到e是否可达，也即e是否i的邻居
